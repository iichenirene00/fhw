**What is the difference between git fetch and git pull?**

Git fetch downloads updates from the remote but doesn’t apply them to branch, while git pull downloads and applies the updates.

**What is the difference between git merge and git rebase? Pro and Cons? **

Git merge combines changes, while git rebase moves changes to the end of the branch.

**How do you resolve merge conflicts in Git?**

The way to resolve merge conflicts is to edit the conflicted file. 

**What is the purpose of .gitignore?**

The .gitignore file tells Git to skip tracking certain files.

**How do you undo a commit that has already been pushed?**

Use git revert to make a new commit or use git reset and push --force to remove the whole thing.

**Can you give me some common git commands?**

Some common git commands is git init, git add, git commit, git push, git checkout, and git branch
